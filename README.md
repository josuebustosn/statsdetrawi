# TrawiStats

Monitor de crecimiento de seguidores para Instagram, diseñado para ser minimalista, hermoso y útil.

![TrawiStats Screenshot](/public/window.svg)

## Características

-   **Contador en Tiempo Real**: Actualizado vía Apify (cada 2 horas).
-   **Calendario de Crecimiento**: Visualización diaria del rendimiento.
-   **Proyecciones**: Estimación de cuándo se alcanzarán los hitos (1k, 5k, 10k).
-   **Calculadoras**: Herramientas para calcular inversión necesaria (CPF).
-   **Modo Oscuro/Claro**: Interfaz adaptable.
